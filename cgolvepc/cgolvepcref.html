Color Game Of Life Visual Exhibition

Pattern Collection Reference

Many interesting Game of Life patterns have been omitted because of the web-based nature of this collection. The patterns which were selected for inclusion here are organized into groups and subgroups to make file transfers small and navigation fast.

Download a copy of the CGOLVE pattern collection (about 250K). Enables viewing of the collection without the internet and without the subgroups. Unzip the distribution and double click the resulting cgolvepc.html file to view the collection.

Download a copy of the tools used to create the CGOLVE pattern collection (about 80K). Enables creation and editing of pattern collections, including the CGOLVE pattern collection. Provides a Windows tool (with C++ source) which creates a new pattern collection from a directory of pattern files. Unzip the distribution and double click the resulting pcauthor.html file to get started.

All files in the distributions may be modified and reproduced without restriction.
